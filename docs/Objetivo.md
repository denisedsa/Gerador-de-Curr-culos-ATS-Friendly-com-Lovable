# Objetivo do Projeto

## Visão Geral

O **Gerador de Currículos ATS Friendly com Lovable** foi desenvolvido como atividade prática para explorar o uso de Inteligência Artificial Generativa e Vibe Coding na criação de aplicações funcionais, utilizando a plataforma Lovable.

O projeto tem como objetivo simplificar a criação de currículos profissionais compatíveis com sistemas **ATS (Applicant Tracking System)**, amplamente utilizados por empresas para realizar a triagem automática de candidatos em processos seletivos.

Além da geração do currículo, o projeto busca proporcionar uma experiência intuitiva ao usuário por meio de uma interface organizada, visualização em tempo real e gerenciamento do histórico de currículos criados.

---

# Objetivos Específicos

* Desenvolver uma aplicação para criação de currículos ATS Friendly.
* Permitir o preenchimento manual das informações profissionais.
* Possibilitar a importação de currículos existentes para edição.
* Exibir uma visualização em tempo real do currículo durante o preenchimento.
* Armazenar o histórico de currículos criados.
* Permitir edição e exclusão de currículos.
* Exportar o currículo em formato PDF mantendo um layout compatível com sistemas ATS.
* Aplicar boas práticas de experiência do usuário (UX) durante toda a navegação.

---

# Motivação

Muitos profissionais encontram dificuldades para elaborar currículos organizados e compatíveis com sistemas de recrutamento automatizados.

Este projeto demonstra como ferramentas de Inteligência Artificial podem acelerar o desenvolvimento de soluções voltadas para esse cenário, permitindo criar aplicações completas por meio da definição de requisitos, engenharia de prompts e refinamento contínuo da solução.

---

# Público-Alvo

O sistema foi idealizado para atender:

* Profissionais em busca de recolocação no mercado.
* Estudantes em início de carreira.
* Pessoas que desejam modernizar seus currículos.
* Recrutadores interessados em currículos padronizados.
* Desenvolvedores interessados em aplicações criadas com IA Generativa.

---

# Resultados Esperados

Ao final do desenvolvimento, espera-se disponibilizar uma aplicação capaz de:

* Facilitar a criação de currículos profissionais.
* Melhorar a compatibilidade dos currículos com sistemas ATS.
* Reduzir o tempo necessário para elaboração de um currículo.
* Demonstrar a utilização prática de IA Generativa no desenvolvimento de software.
* Servir como projeto de estudo e portfólio para apresentação de competências em documentação, engenharia de prompts e desenvolvimento assistido por Inteligência Artificial.
